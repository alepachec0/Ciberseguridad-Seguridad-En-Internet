Tipos De Amenazas

Tipo de Amenaza Descripción
Phishing Engaños para robar datos personales a través de correos o sitios falsos.
Malware Software diseñado para causar daño, robar o espiar información.
Ransomware Bloquea el acceso a archivos hasta pagar un rescate.
Spyware Programa espía que recopila información sin permiso.
Ataques DDoS Colapsan servicios saturándolos de tráfico desde muchas fuentes.
Ingeniería Social Manipula a personas para obtener información confidencial.
